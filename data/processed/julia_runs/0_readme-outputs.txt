Readme outputs.


Filenames have the experiment number followed by 4 numbers separated by "_".
	First number is the treatment number, 1-10.
	Second number is the Sv (survivorship) treatment, 0-1.
	Third number is the BB (birth beta,fecundity) treatment, 0-1.
	Fourth number is the Dt (drift) treatment, 1-0.1.

Example: exp_1-1_1_1_0.5
